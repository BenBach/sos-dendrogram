Manual

This plug-in allows you to play the contents of your SOM on multiple speakers simultaniously using stereo soundcards installed on your system.

Please follow the instructions below.

0 Command line parameters

Make sure that you have set the parameters p_decode and decodeDir to the desired values.

1 Layout and position of speakers

You have to fill in the position of each speaker into the layout table.

  • Enter the desired number of columns and rows into the text fields.
  • Press Gen! to generate the layout table.
  • For each cell: enter the string identifying the speaker at this positon.
    Examples:
    • Left speaker of the output line #2 (as shown in the list):
      2L
    • Right speaker of line #0: 0R
    • The string is case insensitive (0r would be valid too.)
  • You can use the buttons Find me (loop) and Find me to play soundclips so that you easily can identify which speaker belongs to which output line.

2 Assignment of SOM units

Next you assign one or more SOM units to each cell of the layout table.

Optionally you can make the application do the assignment automatically. To do so, hit Automatic assignment. If you prefer to do this manually or if you want to correct an automatic assignment, follow these steps:

  • Select one or more cells of the layout table.
  • Select one or more units in the SOMPane.

If you select a layout table cell the units assigned to it will be highlighted. If you select a range of cells the union of the assigned units will be highlighted.

3 Save / load assignment

You can save your current speaker layout and assignment to a file which you can load at a later point of time. Use buttons Save and Load, respectively.

Note that the order of the speaker in the list might change if you restart your computer or if you dis- and re-connect a (usb-)speaker.

4 Playback

If you are sure that the layout table, the assignment of the SOM units and the preferences have been set correctly, start playback by pressing Start playback
